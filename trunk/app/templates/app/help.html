{% trans "Logging in" %}

{% blocktrans %} To access Transfr, you must first log in. To do so, enter the username and the password sent to you by an administrator in the login form. (Fig. 1) {% endblocktrans %}

Fig. 1

{% trans "The interface" %}

{% blocktrans %} If the username and password you entered are valid, you will be taken to the main screen of Transfr: {% endblocktrans %}

Fig. 2

  • 1{% trans "Menu" %}

    {% blocktrans %} The Files button will take you to the main Transfr page, which contains a listing of uploaded files and the Logout button will terminate your session. {% endblocktrans %}

  • 2{% trans "Help and About Transfr, language switcher" %}

    {% blocktrans %} About Transfr gives information on this software, who built it and why. The help button links to this page and finally the language switcher allow you to switch the interface language. {% endblocktrans %}

  • 3{% trans "File list with actions (delete, upload)" %}

    {% blocktrans %} The file list show uploaded files, their names, size and upload date. The Delete selection button will delete permanently all the selected files. The Upload a file button shows the file upload form (fig. 3). {% endblocktrans %}

    {% trans "Each files has its own delete and download button." %}

  • 4{% trans "Preview" %}

    {% blocktrans %} When possible, a thumbnail of the selected file will be shown. The selected file can be identified by the color of its row in the file list (pale blue). {% endblocktrans %}

{% trans "Uploading a file" %}

Fig. 3

  • 1{% trans "Click on the Upload a file button" %}

    {% blocktrans %} The file upload form will take the place of the file preview pane. {% endblocktrans %}

  • 2{% trans "Click Browse in the file field" %}

    {% blocktrans %} A window will show up to allow you to select the file you want to upload from your computer (fig. 4). {% endblocktrans %}

  • 3{% trans "Optional: enter a comment about the file" %}

    {% blocktrans %} The file description will show up in the file list, below the file name. {% endblocktrans %}

  • 4{% trans "Click on the Save button" %}

    {% blocktrans %} The file upload will then begin; if the file is large, a progressbar will indicate the upload progress. When the upload is complete, you will be automatically redirected to the file list, where your uploaded file will be listed. {% endblocktrans %}
